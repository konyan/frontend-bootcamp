🏷️ Document Structure & Metadata

This page demonstrates proper HTML head configuration for:

  • Character encoding (UTF-8)
  • Responsive viewport
  • SEO meta tags
  • Open Graph tags for social sharing
  • Twitter Card tags
  • Favicon and theme color

Head Section Breakdown

Character Encoding

<meta charset="UTF-8">

Must be first in head. Tells browser to interpret text as UTF-8 (supports all languages).

Responsive Viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Critical for mobile. Tells browser to scale content to device width.

SEO Tags

<title>
Shows in browser tab and search results. Keep under 60 characters.
<meta name="description">
Shows in search results. Keep under 160 characters. Include target keywords.
<meta name="keywords">
Hints to search engines. Less important now, but still useful.

Open Graph Tags

Control how page appears when shared on Facebook, LinkedIn, Slack, Discord, etc.

og:title
Title shown in preview
og:description
Description shown in preview
og:image
Image shown in preview (1200x630px recommended)
og:url
Canonical URL of the page
og:type
Content type: article, website, video, etc.

Brand & Browser

favicon
Small icon shown in browser tab. Improves brand recognition.
theme-color
Color for browser UI and mobile app shortcuts.

Why This Matters

SEO Impact

Proper head configuration directly affects search engine rankings. Missing or poor metadata reduces visibility. Good metadata:

  • Improves click-through rate from search results
  • Helps search engines understand page content
  • Supports rich snippets (ratings, reviews, etc.)

Social Sharing

When someone shares your page, Open Graph and Twitter tags control the preview. A professional, branded preview gets more clicks than a generic one.

Mobile UX

The viewport meta tag and theme-color make your site look professional on mobile. Without it, pages render zoomed out and are hard to read.

Quick Checklist

  • ✅ UTF-8 charset (first in head)
  • ✅ Viewport meta tag
  • ✅ Meaningful title (under 60 characters)
  • ✅ Descriptive meta description (under 160 characters)
  • ✅ Open Graph tags (og:title, og:description, og:image)
  • ✅ Twitter Card tags (if shared on Twitter)
  • ✅ Favicon
  • ✅ Theme color for mobile
  • ✅ Valid HTML (run through W3C validator)
