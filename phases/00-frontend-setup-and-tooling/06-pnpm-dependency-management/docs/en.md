# pnpm and Dependency Management

> Dependency installs are one of the easiest places for a team to drift apart.

**Type:** Build
**Languages:** JavaScript
**Prerequisites:** None
**Time:** ~60 minutes

## Learning Objectives

- Understand why pnpm is popular for modern frontend repos
- Install dependencies with a strict lockfile workflow
- Use workspace-aware package management effectively
- Recognize when dependency issues come from the manager, not the code

## The Concept

A good package manager balances speed, disk efficiency, and reproducibility.

## Build It

### Step 1
Install pnpm with Corepack or your preferred method.

### Step 2
Inspect the lockfile and dependency tree for one demo project.

### Step 3
Try a workspace install if you have multiple packages.

### Step 4
Compare how pnpm handles hoisting versus npm or yarn.

## Use It

Apply the workflow to your current frontend project instead of waiting until later.

## Exercises

1. Initialize a tiny frontend app with pnpm.
1. Create a workspace and add one shared dependency.
1. Explain why lockfiles matter in a team setting.
