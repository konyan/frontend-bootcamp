# Phase 0: Frontend Setup & Tooling

> Build a professional frontend workflow: shell, editor, runtimes, git, compilers, and deployment.

This phase covers the habits and tools that keep modern frontend work fast and predictable. By the end you will have a configured shell, a productive VS Code setup, Node version management, pnpm for dependencies, a structured git workflow, TypeScript configured correctly, and a working deploy pipeline.

Inspired by: [A Modern Front-end Project Setup](https://medium.com/@mihirsanghavi/a-modern-front-end-project-setup-98e698c71aad) — updated for 2024 tooling.

See [ROADMAP.md](../../ROADMAP.md) for status tracking.
